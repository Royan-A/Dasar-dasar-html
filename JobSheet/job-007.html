PENGGUNAAN TAG-TAG PADA HTML


      Untuk memperoleh suatu tampilan yang menarik dalam dokumen HTML, diperlukan kemampuan khusus salah satunya yaitu kemampuan untuk memformat/mengatur pada teks pada dokumen HTML

      Untuk melakukan pengaturan teks tidak terlepas dari penggunaan tag-tag HTML Seperti:

1. Tag <i> tulisan </i> : Untuk Menampilkan tulisan miring
2. Tag <strike> tulisan </strike> : Untuk Menampilkan tulisan bergaris tengah
3. Tag <b> tulisan </b> : Untuk Menampilkan tulisan tebal
4. Tag <small> tulisan </small> : Untuk Menampilkan tulisan lebih kecil dari ukuran normal
5. Tag <big> tulisan </big> : Untuk Menampilkan tulisan lebih besar dari ukuran normal
6. Tag <tt> tulisan </tt> : Untuk Menampilkan tulisan seperti tulisan mesin ketik
7. Tag <u> tulisan </u> : Untuk Menampilkan tulisan bergaris bawah
8. Tag <sub> tulisan </sub> : Untuk Menampilkan tulisan menjadisubscript
9. Tag <sup> tulisan </sup> : Untuk Menampilkan tulisan menjadisupscript
